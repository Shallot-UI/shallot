# Line Height

Line height is the vertical space between lines of text. They are defined in the theme as an object of keys and numbers. Shallot expects there to exist keys corresponding to standard line heights like `xs`, `sm`, `md`, `lg`, and `xl`. You can also define your own keys and values when you call `makeTheme()`.

## Default Values

Below are the default values for the line heights in the Shallot theme.

<Canvas>
  <ShallotProvider>
    <Column maxUnitWidth={25}>
      {['sm', 'md', 'lg', 'xl'].map((lineHeight) => (
        <Text
          key={lineHeight}
          unitsAround={1}
          shallot={{
            lineHeight: getLineHeight(lineHeight),
            fontFamily: getFontFamily('Body'),
            fontSize: getFontSize('md'),
          }}
        >
          This is the default line height for "{lineHeight}". We've extended
          this paragraph to demonstrate the line height.
        </Text>
      ))}
    </Column>
  </ShallotProvider>

### Custom Line Heights

You can override the default line heights by passing an object to the `lineHeights` key of the theme object passed to the `theme` prop of the `ShallotProvider` component. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = {
  lineHeights: {
    xxs: 1.05, // define a new line height
    md: 1.5, // override a default line height
  },
}

<ShallotProvider theme={myTheme}>
  ...// Your app
</ShallotProvider>
```

To apply a line height to a `Text` component, use the `lineHeight` prop.

```jsx
<Text lineHeight="md">Some text with medium line height</Text>
```
