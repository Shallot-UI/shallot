# Elevations

Elevations are used to give depth and shadow to components such as `Row`, `Column`, and `Box`. They are defined in the theme as an object of keys and values, where each key corresponds to a specific elevation state and the value is an object that defines the `boxShadow` and `transform` properties.

## Default Values

Below are the default values for the elevations in the Shallot theme.

<Canvas>
  <ThemeProvider theme={makeTheme()}>
    <Column backgroundColor="Shading.125" radius="xl">
      <Row alignCenter unitsAround={4}>
        {['pressable', 'hover', 'pressed', 'focused', 'none'].map(
          (elevation) => (
            <Column
              key={elevation}
              elevation={elevation}
              shallot={{
                backgroundColor: 'Shading.100',
                borderRadius: getRadius('lg'),
                height: getUnits(8),
                width: getUnits(8),
                margin: getUnits(1),
                alignItems: 'center',
                justifyContent: 'center',
              }}
            >
              {console.log(getElevation(elevation))}
              <Text
                shallot={{
                  // typeface: "Monospace",
                  fontSize: getFontSize('sm'),
                  textColor: getColorShade('Shading.350'),
                }}
              >
                {elevation}
              </Text>
            </Column>
          ),
        )}
      </Row>
    </Column>
  </ThemeProvider>
</Canvas>

- `pressable`: The default elevation state for a pressable component.
- `hover`: The elevation state when the component is being hovered over.
- `pressed`: The elevation state when the component is being pressed.
- `focused`: The elevation state when the component is being focused.
- `none`: The elevation state when the component has no elevation.

## Customizing Elevations

You can customize the default elevations by passing an object to the `elevations` key of the theme object passed to `makeTheme()`. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = makeTheme({
  elevations: {
    pressable: {
      transform: 'translateY(0)',
      boxShadow: '0 4px 6px -4px rgba(0,0,0,0.4)',
    },
    hover: {
      boxShadow: '0 4px 12px -4px rgba(0,0,0,0.4)',
      transform: 'translateY(-4px)',
    },
  },
})
```

To apply an elevation to a component, use the `elevation` prop.

```jsx
<Box elevation="pressable">This box has the "pressable" elevation.</Box>
```
