# Shadow

Shadow (box-shadow) is used to give elevation & depth to components. They are defined in the theme as an object of keys and values, where each key corresponds to a specific shadow state and the value is an object that defines the `boxShadow` and `transform` properties.

## Default Values

Below are the default values for the shadows in the Shallot theme.

<Canvas>
  <ShallotProvider>
    <Column
      shallot={{
        backgroundColor: getColor('Shading', 125),
        borderRadius: getRadius('xl'),
      }}
    >
      <Row alignCenter unitsAround={4}>
        {[100, 200, 300, 400, 500, 600, 700, 800, 900, 'focused', 'none'].map(
          (shadowState) => (
            <Column
              key={shadowState}
              shallot={{
                boxShadow: getShadow(shadowState),
                backgroundColor: getColor('Shading', 100),
                borderRadius: getRadius('lg'),
                height: getUnits(8),
                width: getUnits(8),
                margin: getUnits(1),
                alignItems: 'center',
                justifyContent: 'center',
              }}
            >
              <Text
                shallot={{
                  fontFamily: getFontFamily('Monospace'),
                  fontSize: getFontSize('sm'),
                  textColor: getColorShade('Shading.350'),
                }}
              >
                {shadowState}
              </Text>
            </Column>
          ),
        )}
      </Row>
    </Column>
  </ShallotProvider>
</Canvas>

- `pressable`: The default box-shadow for a pressable component.
- `hover`: The box-shadow when the component is being hovered over.
- `pressed`: The box-shadow when the component is being pressed.
- `focused`: The box-shadow when the component is being focused.
- `none`: The box-shadow when the component has no shadow set.

## Customizing Shadows

You can customize the default shadow by passing an object to the `shadows` key of the theme object passed to the `ShallotProvider` component. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = makeTheme({
  shadow: {
    pressable: {
      transform: 'translateY(0)',
      boxShadow: '0 4px 6px -4px rgba(0,0,0,0.4)',
    },
    hover: {
      boxShadow: '0 4px 12px -4px rgba(0,0,0,0.4)',
      transform: 'translateY(-4px)',
    },
  },
})
```

To apply a shadow to a component, use the `shadow` prop.

```jsx
<Box shadow="pressable">This box has the "pressable" shadow.</Box>
```
