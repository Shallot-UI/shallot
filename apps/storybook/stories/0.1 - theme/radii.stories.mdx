# Radii

Radii (the plural of radius) are used to round the corners of elements. They are defined in the theme as an of oject of keys and numbers. Shallot expects there to exist keys corresponding to standard t-shirt sizes `sm`, `md`, `lg`, and `xl`, as well as a special radius called `pill`. The pill radius is used to create pill-shaped buttons and elements. You can also define your own keys and values when you call `makeTheme()`.

## Default Values

Below are the default values for the radii in the Shallot theme.

<Canvas>
  <ThemeProvider theme={makeTheme()}>
    <Column
      shallot={{
        backgroundColor: getColor('Shading', 125),
        borderRadius: getRadius('xl'),
      }}
    >
      <Row shallot={{ alignItems: 'center', margin: getUnits(2) }}>
        {['sm', 'md', 'lg', 'xl', 'pill'].map((radius) => (
          <Column
            key={radius}
            shallot={{
              backgroundColor: getColor('Shading', 100),
              borderColor: getColor('Shading', 250),
              borderRadius: getRadius(radius),
              borderWidth: 1,
              height: getUnits(5),
              width: getUnits(5),
              margin: getUnits(1),
              alignItems: 'center',
              justifyContent: 'center',
            }}
          >
            <Text
              shallot={{
                // typeface: 'Monospace',
                fontSize: getFontSize('sm'),
                textColor: getColor('Shading', 350),
              }}
            >
              {radius}
            </Text>
          </Column>
        ))}
      </Row>
    </Column>
  </ThemeProvider>
</Canvas>

### Custom Radii

You can override the default radii by passing an object to the `radii` key of the theme object passed to `makeTheme()`. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = makeTheme({
  radii: {
    xs: 2, // define a new radius
    md: 4, // override a default radius
  },
})
```
