# Letter Spacing

Letter spacing is the amount of space between letters in a text element. They are defined in the theme as an object of keys and numbers. Shallot expects there to exist keys corresponding to standard spacings like `sm`, `md`, and `lg`. You can also define your own keys and values when you call `makeTheme()`.

## Default Values

Below are the default values for the letter spacings in the Shallot theme.

<Canvas>

          key={letterSpacing}
          shallot={{
            margin: getUnits(1),
            letterSpacing: getLetterSpacing(letterSpacing),
            typeface: getTypeface('Body'),
            fontSize: getFontSize('md'),
          }}
        >
          Default letter spacing for "{letterSpacing}".
        </Text>
      ))}
    </Column>
  </ShallotProvider>

### Custom Letter Spacings

You can override the default letter spacings by passing an object to the `letterSpacings` key of the theme object passed to `makeTheme()`. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = makeTheme({
  letterSpacings: {
    xs: 0.5, // define a new letter spacing
    md: 1, // override a default letter spacing
  },
})
```

To apply a letter spacing to a `Text` component, use the `letterSpacing` prop.

```jsx
<Text letterSpacing="md">Some text with medium letter spacing</Text>
```
