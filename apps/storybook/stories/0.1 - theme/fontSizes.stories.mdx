# Font Sizes

Font sizes determine the size of text in your design system. They are defined in the theme as an object of keys and numbers. Shallot expects there to exist keys corresponding to standard sizes like `xs`, `sm`, `md`, `lg`, and `xl`. You can also define your own keys and values when you call `makeTheme()`.

## Default Values

Below are the default values for the font sizes in the Shallot theme.

<Canvas>
  <ShallotProvider>
    <Column>
      {['xs', 'sm', 'md', 'lg', 'xl'].map((fontSize) => (
        <Text
          key={fontSize}
          shallot={{
            // fontFamily: getFontFamily("Body"),
            margin: getUnits(1),
            fontSize: getFontSize(fontSize),
          }}
        >
          Default size for "{fontSize}".
        </Text>
      ))}
    </Column>
  </ShallotProvider>
</Canvas>

## Custom Font Sizes

You can override the default font sizes by passing an object to the fontSizes key of the theme object passed to `makeTheme()`. You don't have to define all of the keys, only the ones you want to override.

```ts
const myTheme = makeTheme({
  fontSizes: {
    xxs: 8, // define a new font size
    md: 18, // override a default font size
  },
})
```
