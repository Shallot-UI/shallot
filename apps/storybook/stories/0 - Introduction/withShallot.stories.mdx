# `withShallot()`

The `withShallot` functions allows you to easily build a styled components that supports the `shallot` prop and a variant. This is useful for building components that are not included in the Shallot library.

## Basic Usage

```tsx
import { withShallot } from '@shallot-ui/web'

const Container = withShallot('div', {
  backgroundColor: getColor('Shading', 200),
  borderRadius: getRadius('lg'),
})
```

## Variants

```tsx
import { withShallot } from '@shallot-ui/web'

const Container = withShallot(
  'div',
  {},
  {
    scope: 'Box',
    variant: 'Container',
  },
)
```

Now your component will have a `Box.Container` variant by default, and the variant can be overridden while preserving the default scope.

```tsx
// Will render with the variant `Box.Wrapper`
<Container variant="Wrapper" />
```

## With Layout props

If you want to enable Shallot layout props on your component, you can use the `withBoxShallot` function instead. The only difference is that this function requires a `flexDirection` property to be passed in the default styles object.

```tsx
import { withBoxShallot } from '@shallot-ui/web'

const Container = withBoxShallot('div', {
  flexDirection: 'column',
  backgroundColor: getColor('Shading', 200),
  borderRadius: getRadius('lg'),
})
```

Now you can use layout props to style the container component.

```tsx
import { withTextShallot } from '@shallot-ui/web'

const Heading = withTextShallot('h1', {
  fontFamily: getFontFamily('Body'),
  color: getColor('Shading', 500),
  fontSize: getFontSize('xl'),
  fontWeight: 700,
})
```

If you want to enable Shallot text props on your component, you can use the `withTextShallot` function instead. The only difference is that this function requires a `fontSize` property to be passed in the default styles object.
